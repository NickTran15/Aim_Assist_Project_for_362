# Aim Assist

A game that is used to help improve your aim for first person shooter games. This game includes 3 game modes all
targeted towards helping improve different aspects of your gameplay, such as accuracy, reaction time, and speed.

## Gameplay
Choose from three different gamemodes
- Freemode
  - In Freemode one target spawns at a time and the only objective is to keep your accuracy percentage as high as possible.
- Random
  - Random tests the players reaction time and mechanics to see how long it takes to click 10 targets
- Rush
  - Rush challenges the players speed by consistently spawning targets for 30 seconds and seeing how many targets the player is able to click within that time frame
